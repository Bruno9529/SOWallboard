# Wallboard Dashboard

A real‑time wallboard dashboard built with HTML, CSS and JavaScript.  
Designed to display live agent activity, call statistics, and availability across departments.

## 🔍 Features

- Department‑based agent grouping
- Real‑time call metrics via WebSocket
- Live agent status (Available, Busy, Ringing, DND, Offline)
- Support availability alert
- Compact / detailed view toggle
- Status breakdown toggle
- Auto‑reconnect handling
- Clock integrated into the top bar
- TV / wallboard‑friendly layout
